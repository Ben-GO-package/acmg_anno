# BS2 - 与疾病表型不符的纯合变异

判断条件:
1. 纯合检查:
   - 对于晚发病基因:
     - 纯合数 >= 5判定为BS2
   - 对于其他基因:
     - 纯合数 > 0判定为BS2
   - 检查数据来源:
     - ExAC HomoAlt Count
     - GnomAD HomoAlt Count

2. 显性遗传模式特殊检查:
   - 对于AD/PD/YL遗传方式
   - 至少2个人群频率数据库中有频率记录
   - 检查数据库:
     - GnomAD EAS AF
     - GnomAD AF
     - 1000G AF
     - ESP6500 AF
     - ExAC EAS AF
     - ExAC AF

注意事项:
- 区分晚发病基因和其他基因
- 对显性遗传有特殊判断标准
